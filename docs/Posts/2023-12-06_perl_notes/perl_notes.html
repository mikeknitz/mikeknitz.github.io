## Hello World

### Script

- `print()`
- `say()`: appends a newline
  
```{perl}
#!/usr/bin/env perl
use strict; use warnings;

say "Perl version: $]";

my $var = "Hello there";
print $var;
print $var;
print "\n";
say $var;
```

### One-liner

```{bash}
#!/bin/bash

# E = enable optional features
perl -e 'print "Hello there";'
perl -E 'say "Hello there";'
perl -e 'print "Hello there";'
```
