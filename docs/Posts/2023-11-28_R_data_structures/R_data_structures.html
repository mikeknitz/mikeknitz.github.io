Subsetting

Subsetting with [] returns a list

a <- list(a = 1, b = 2, c = 3)
a_sub1 <- a[1:2]
a_sub1
$a
[1] 1

$b
[1] 2
a_sub2 <- a[-2]
a_sub2
$a
[1] 1

$c
[1] 3

Subsetting with [[]] for a single element returns the element, not an object of class “list”

one_item <- a[[2]]
one_item
class(one_item)
[1] 2
[1] "numeric"

## Lists

### Create lists

```{r}
my_list <- list(a = 1, b = 2, c = c("a", "b", "c"))

my_list
```

```{r}
my_list <- c("a", "b", "c") |>
    as.list()

my_list
```

```{r}
my_list <- c("name1" = "a", "name2" = "b", "name3" = "c") |>
    as.list()

my_list
```

### Concatenate lists

```{r}
a <- list(a = 1, b = 2, c = 3)
b <- list(d = 4, e = 5, f = 6)

c <- c(a, b)
c
str(c)
class(c)
```

### Subsetting

**Subsetting with `[]` returns a list**

```{r}
a <- list(a = 1, b = 2, c = 3)
a_sub1 <- a[1:2]
a_sub1
```

```{r}
a_sub2 <- a[-2]
a_sub2
```

**Subsetting with [[]] for a single element returns the element, not an object of class "list"**

```{r}
one_item <- a[[2]]
one_item
class(one_item)
```
